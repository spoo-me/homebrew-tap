# spoo-me Homebrew Tap

Homebrew formulae for [spoo.me](https://spoo.me) tools.

## Install

```bash
brew install spoo-me/tap/spoo
```

Formulae in this tap are published automatically by [GoReleaser](https://goreleaser.com) from [spoo-me/spoo-cli](https://github.com/spoo-me/spoo-cli) releases — they are not edited by hand.
